# Grant Application Templates

## Standard Sections

1. **Executive Summary** — One-pager: problem, solution, market, ask
2. **Technical Specification** — For SBIR/STEM grants
3. **Budget** — CAPEX/OPEX breakdown with timeline
4. **Team** — Human + AI workforce overview
5. **Traction** — Milestones, pilots, LOIs
6. **Competitive Landscape** — SWOT analysis
7. **Use of Funds** — Detailed deployment plan
